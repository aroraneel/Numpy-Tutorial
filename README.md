# NumPy Learning Notebook

A beginner-friendly NumPy project that covers the fundamental concepts of NumPy through practical examples, detailed explanations, and well-structured Jupyter Notebook cells.

This repository was created as part of my Artificial Intelligence and Machine Learning learning journey to build a strong foundation in numerical computing using Python.

---

## Features

* Beginner-friendly explanations
* Well-structured Jupyter Notebook
* Step-by-step practical examples
* Clean and readable code
* Markdown explanations for every topic
* Additional examples beyond the original tutorial
* Python script version included for quick reference

## Installation

Clone the repository:

```bash
git clone https://github.com/aroraneel/Numpy-Tutorial
```

Navigate to the project directory:

```bash
cd NumPy
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage

Open the notebook using Jupyter Notebook:

```bash
jupyter notebook
```

or JupyterLab:

```bash
jupyter lab
```

To run the Python version:

```bash
python scripts/numpy.py
```
